TMDB API-Key einrichten

Um die echten Filmcover zu laden, brauchst du einen kostenlosen API-Key von The Movie Database (TMDB).

1. Erstelle einen Account auf themoviedb.org
2. Gehe zu Einstellungen → API
3. Beantrage einen API-Key (Developer)
4. Kopiere den "API Key (v3 auth)" hier rein
